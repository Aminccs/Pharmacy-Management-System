نظام إدارة الصيدلية | Pharmacy Management System 💊
هذا هو مشروعي الأول في لغة Java، وهو عبارة عن نظام متكامل لإدارة الأدوية والمخزون في الصيدلية، تم تطويره باستخدام مفاهيم البرمجة كائنية التوجه (OOP).

🚀 المميزات (Features)
إضافة الأدوية: إدخال بيانات الأدوية (الاسم، السعر، النوع).

إدارة المخزون: متابعة الكميات والتأكد من عدم تجاوز السعة الاستيعابية.

معالجة الأخطاء: استخدام الـ Exceptions المخصصة مثل InvalidDataException لضمان صحة البيانات.

أنواع الأدوية: استخدام الـ Enums لتصنيف الأدوية (TABLET, CAPSULE, VITAMIN).

🛠️ التقنيات المستخدمة (Tech Stack)
Language: Java

IDE: Eclipse

Version Control: Git & GitHub

📂 هيكلة المشروع (Structure)
Medicine.java: الكلاس الأساسي لتمثيل بيانات الدواء.

PharmacyManager.java: المسؤول عن العمليات والمنطق البرمجي (Logic).

PharmacyApp.java: الواجهة البرمجية للتفاعل مع المستخدم (Main Menu).
